潘仰祐的學習歷程檔案備份

　　各位！這就是我（潘仰祐）的學習歷程檔案備份了。你也想要備一份嗎？來 這裡 下載備份器ㄅ！（前提是我寫的 Code 還能跑啦w）

　　點擊每個檔案的「嘗試預覽」按鈕，可以在瀏覽器中檢視該檔案。若是瀏覽器不支援，也可以直接在資料夾中找到該檔案，並且開啟。

請稍待，本檔案正由您的瀏覽器渲染成方便檢視的表格。
　　在建中的有緣人啊，你竟然看到這裡了！如果還來得及傳多元表現，（你卻發現駝課展圖已經不讓你傳自主學習計劃了 && 你願意花時間偽造一份自主學習計劃）or 以前的計劃老師忘記審核了，你不如試試看這個東西。這對乖寶寶不太公平，所以請只告訴你信得過／你認為有需要的同學——千萬不要告訴老師。
　　當然，我敢把這個放出來，也是因為我不相信趕出來的可以比慢慢做的強 :P。
簡而言之：把 自主學習破解器 拉到書籤列。去 駝課展圖 -> 自主學習 -> 選學期，點書籤列的「自主學習破解器」，然後開始編輯。
  1. 把 自主學習破解器 拖入書籤列。
  2. 去 駝課展圖
  3. 點「自主學習」。
  4. 選定任何一個點了會顯示「今天不在期限內」的學期。請不要選擇那些會顯示「尚未設定期限」的學期。
  5. 點擊書籤列中的「自主學習破解器」。
  6. 開始編輯。編輯後請記得按存檔。如果按存檔後，存檔按鈕消失，請再重新按一次「自主學習破解器」
  7. 全部完成後，請按最左方的「匯出自主學習」並選擇學年度與學期，然後匯出。
